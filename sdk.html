Quick Start 5 MIN SETUP
Get NexAuth running in your app in 3 simple steps.
1
Create an Application
Go to Dashboard → Applications → New App. Copy your App ID.
2
Add NexAuth SDK to your project
Download the SDK file for your language below and add it to your project. No external dependencies except HTTP client.
3
Initialize & Authenticate
Create a NexAuth instance with your App ID and call Login() or CheckLicense().
Login AUTH
Authenticate a user with username and password. Returns a JWT token on success.
POST /api/auth/login
ParameterTypeRequiredDescription
appIdstringrequiredYour Application ID from dashboard
usernamestringrequiredUser's username or email

NexAuth.cs — Login
// Initialize NexAuth with your App ID
var auth = new NexAuth("YOUR_APP_ID");

NexAuth.java — Login

    public void onSuccess(JSONObject user) {
        System.out.println("Logged in: " + user.getString("username"));

    @Override
    public void onFailure(String error) {

NexAuth.cs — License Check
var auth = new NexAuth("YOUR_APP_ID");
string hwid = NexAuth.GetHWID(); // Auto hardware fingerprint

bool valid = await auth.CheckLicense(licenseKey, hwid);

if (valid)
{
    MessageBox.Show("License activated! Enjoy the app.");
    new MainForm().Show();
}
else
{
    MessageBox.Show("Invalid or expired license key.");
    Application.Exit();
}

HWID Lock SECURITY
Hardware ID locking binds a license or account to a specific machine. Prevents key sharing.
⚠️
Once HWID is bound, the key can only be used on that machine. Admin can reset HWID from Dashboard → Users.
C#
C++
Python
Java
JavaScript
VB.NET
NexAuth.cs — HWID
// Get unique hardware fingerprint
string hwid = NexAuth.GetHWID();
// Returns: CPU ID + Motherboard ID + Disk Serial (MD5 hashed)

// Pass to login or license check
bool ok = await auth.Login(username, password, hwid);

// Or check HWID ban status
bool banned = await auth.IsHWIDBanned(hwid);
if (banned) { Application.Exit(); }
nexauth.cpp — HWID
std::string hwid = NexAuth::getHWID();
bool banned = auth.isHWIDBanned(hwid);
if (banned) std::exit(1);

NexAuth.java — HWID

nexauth.js — HWID

Error Codes REFERENCE
All API errors return a JSON body with success: false and a message.
HTTP CodeErrorMeaning
400Bad RequestMissing required fields in body
401UnauthorizedInvalid credentials or expired token
403ForbiddenAccount banned or HWID mismatch
404Not FoundApp ID, user, or license not found
409ConflictUsername or email already taken
429Too Many RequestsRate limit hit — wait 15 min
500Server ErrorInternal error — retry later
401 Unauthorized Error Response
{
  "success": false,
  "message": "Invalid credentials"
}
